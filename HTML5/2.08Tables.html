Design

  • Make sure to sketch your layout before you code.
  • Decide on the number of rows and columns.
  • Decide if any row or column will span multiple cells.

Browsers expect same number of cells in each row.So you need to identify which number of row or column to span.

The tags

  • <table>-the container tag.
  • <tr>-the table rows.
  • <td>-the table columns(cells) in each row /(table data)

Table heading

  • How would you add table heading?(The bold cells at top that define the data.)
  • Some people use bold font,we use semantic tags.
  • <th>....</th>--table heading.Instead of table data (td) use Table Heading (th).
